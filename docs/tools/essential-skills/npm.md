# npm 学习笔记

## npm 常用命令

* npm init 初始化
* npm publish 发布
* npm unpublish 取消发布

  ```shell
  npm unpublish [<@scope>/]<pkg>[@<version>]
  ```

* npm deprecate 废弃

  ```shell
  npm deprecate <pkg>[@<version>] <message>
  ```

* npm link 本地 link
* npm config list 配置信息
* .npmignore 忽略不要发布的文件
* prepublishOnly 发布前执行的命令
* npm whoami 登录信息
* npm login 登录
* npm outdated 版本更新情况
* npm view sass versions 查看某一资源线上版本
